Fungsi: Program ini berfungsi untuk menghitung total biaya pengiriman paket berdasarkan tiga faktor: berat paket, jarak tempuh, dan volume paket.
Cara Kerja:Program meminta 5 input dari pengguna: berat (kg), jarak (km), panjang (cm), lebar (cm), dan tinggi (cm).Biaya Jarak dihitung menggunakan if-else. Jika jarak $\le$ 10 km, biayanya Rp 4.250/kg. Jika di atas 10 km, biayanya Rp 6.000/kg.Biaya Volume dihitung dengan mengalikan panjang * lebar * tinggi. Jika volume paket melebihi 100 cmÂ³, akan dikenakan biaya tambahan tetap sebesar Rp 50.000.Total Biaya adalah hasil penjumlahan dari Biaya Jarak + Biaya Volume.

Latihan UKL2
Fungsi: Program sederhana untuk menentukan apakah sebuah bilangan bulat yang dimasukkan oleh pengguna termasuk bilangan ganjil atau genap.
Cara Kerja:
Program meminta 1 input bilangan bulat dari pengguna.
Program menggunakan operator modulus (%) untuk mengecek sisa bagi.
Jika bilangan % 2 hasilnya 0, maka bilangan tersebut adalah GENAP.
Jika hasilnya bukan 0, maka bilangan tersebut adalah GANJIL
